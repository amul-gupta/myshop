# 🛒 MyShop App

A highly scalable and modular backend system for an e-commerce platform.

## 🚀 Features
- User management
- Product management
- Order management
- Notification management
- Inventory management
- Payment management
- AI services


## 🛠️ Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate

### Database
- MySQL

### Build Tool
- Maven

### API & Testing
- RESTful APIs
- Postman

### Architecture
- Layered Architecture (Controller, Service, Repository)
- Microservice-ready design principles

### Tools & Others
- Git & GitHub
- IntelliJ IDEA
